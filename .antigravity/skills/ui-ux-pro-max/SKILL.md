# UI UX Pro Max Skill

An AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## Features
- **67 UI Styles**: Glassmorphism, Minimalism, Bento Grid, etc.
- **96 Color Palettes**: Industry-specific palettes.
- **57 Font Pairings**: Curated typography with Google Fonts.
- **99 UX Guidelines**: Best practices and accessibility rules.

## Usage
Use the search script to get design recommendations:
```bash
python3 scripts/search.py "query" --domain style
python3 scripts/search.py "query" --design-system -p "ProjectName"
```

## Anti-Patterns
- Avoid bright neon colors in professional/banking contexts.
- Avoid harsh animations (> 500ms).
- Avoid low contrast text (min 4.5:1).
